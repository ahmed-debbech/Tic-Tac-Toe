Tic Tac Toe

Tic Tac toe is a small and simple game that was made for fun.\ Details: It was funny for me to program this from scratch, it was made with C and SDLv1.2.

Installation

1) Download the executable for Windows from here. 2) Unzip the file and double click the "Tic Tac Toe 2.1.0.exe".

  • IMPORTANT: DO NOT REPLACE, COPY OR DELETE ANYTHING FROM THE FOLDER.

    Contributing
